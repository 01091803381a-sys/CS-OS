# Class 1. 🖥️ CS-OS: 컴퓨터 구조 및 운영체제 수업 정리

컴퓨터의 기본 작동 원리와 운영체제의 핵심 개념을 학습하고 정리하는 저장소입니다.

---

## 📖 1. 정보의 기초: 데이터와 명령어
컴퓨터가 처리하는 모든 정보는 크게 두 가지로 분류됩니다.

* **데이터(Data)**: 컴퓨터가 이해하는 숫자, 문자, 이미지, 동영상과 같은 **정적인 정보**.
* **명령어(Instruction)**: 데이터를 움직이고 컴퓨터를 실제로 **작동시키는 정보**.

---

## 🛠️ 2. 컴퓨터의 4가지 핵심 부품
컴퓨터를 구성하는 가장 핵심적인 하드웨어 요소입니다.

| 부품 | 역할 | 주요 구성 요소 / 특징 |
| :--- | :--- | :--- |
| **CPU** | 메모리의 명령어를 읽고, 해석하고, 실행함 | **ALU**(연산), **제어장치**(해석/제어), **레지스터**(저장) |
| **메모리(RAM)** | 현재 실행 중인 정보를 저장 | 주소(Address)를 사용해 접근, 휘발성 저장소 |
| **보조기억장치** | 보관할 정보를 장기 저장 | HDD, SSD 등 비휘발성 저장소 |
| **입출력장치** | 외부와의 데이터 교환 | 마우스, 키보드, 모니터 등 (I/O 장치) |

> **메인보드(Motherboard)**: 위 부품들이 장착되는 판이며, 부품 간 데이터 통로인 **시스템 버스(System Bus)**가 존재합니다.
> * **주소 버스**: 주소 전달
> * **데이터 버스**: 명령어 및 데이터 전달
> * **제어 버스**: 제어 신호 전달

---

## 🔢 3. 정보의 단위와 표현

### 🔹 Word (워드)
CPU가 한 번에 처리할 수 있는 데이터의 단위입니다.
* **하프 워드**: Word의 1/2 크기
* **풀 워드**: Word의 기본 크기
* **더블 워드**: Word의 2배 크기

### 🔹 수 표현 (2진법 & 16진법)
* **2진법(0b)**: 0과 1로 표현. 음수는 **2의 보수**(1과 0을 뒤집고 1을 더함)로 표현합니다.
* **16진법(0x)**: 0~9, A~F로 표현. 2진법과의 변환이 쉬워(4bit 단위) 컴퓨터 분야에서 애용됩니다.

---

## 🔠 4. 문자 인코딩과 디코딩
컴퓨터가 문자를 이해하고 출력하는 과정입니다.

1.  **인코딩(Encoding)**: 문자를 0과 1의 코드값으로 변환하는 과정.
2.  **디코딩(Decoding)**: 0과 1의 코드를 다시 문자로 해석하는 과정.

### 대표적인 규격
* **ASCII (아스키 코드)**: 알파벳, 숫자, 특수문자 표현. 7비트 사용 (1비트는 패리티 비트).
* **한글 인코딩**:
    * **완성형**: 글자 마다 코드를 부여 (예: EUC-KR, 2byte 사용).
    * **조합형**: 초성, 중성, 종성을 조합하여 코드 생성.
